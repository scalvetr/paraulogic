# paraulogic

## Install python

Ubuntu
```shell
sudo apt install python3.9
```

Mac
```shell
brew install python@3.9
brew install pyenv
pyenv install 3.9.9
```

## Install dependencies

```shell
python3 -VV
python3 -m site
python3 -m pip install --upgrade unidecode
```

## Run

```shell
# assuming:
# G is mandatory char
# O, E, S, A, L and P are the optional chars
python3 run.py g oesalp
```
